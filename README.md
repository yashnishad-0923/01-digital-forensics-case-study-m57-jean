# M57-Jean Digital Forensics Investigation

## Overview

This repository documents a complete digital forensic investigation of the M57-Jean scenario.

The objective was to determine how a confidential employee spreadsheet left the suspect's laptop and evaluate the claim that the system had been compromised.
